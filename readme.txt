Twitter Fan Box or Follower Box for Wordpress with Facebook Like box style interface and many more customizations.

== Description ==

Twitter Follow Box is built over [Twitter FollowBox jQuery Plugin](http://www.jobyj.in/twitter-follow-box-widget/ "jj-WP Easy Navigation"). It helps wordpress developers / site owners to add facebook like box style widget for twitter. In this beta version, WP Twitter Mega Fan Box Widget supports following options.

1. Settings page accepts your twitter username
2. Width and height of widget can be changed based on your theme size
3. Background color and text color of almost all fields can be changed
4. Footer link can be customized

For more information on various plugins and development, visit [my blog](http://www.jobyj.in/ "Joby's Blog")

[CLICK HERE TO VIEW DEMO](http://www.jobyj.in/ "Joby's Blog")

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
